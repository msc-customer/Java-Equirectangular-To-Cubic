This directory contains files from the SaladoPlayer distribution. They are used for compatibility tests. The files in this directory are released under different licenses - SaladoPlayer itself is GPL, swfobject.js is MIT, and so on. Please make sure you understand which license applies before redistributing any file you find here.

The files here are also not a complete distribution of SaladoPlayer. Please go to: 

    http://panozona.com/wiki/Main_Page 

for more information about SaladoPlayer, and to get a full distribution package.
